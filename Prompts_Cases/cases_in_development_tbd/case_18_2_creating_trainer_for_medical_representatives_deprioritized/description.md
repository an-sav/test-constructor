**2. Шаги выполнения:**

* Собрать материалы по продукту, которые распространялись ранее.
* Выделить наиболее актуальные и используемые материалы.
* Подготовить консолидированную презентацию, используя язык с медицинской направленностью, с основными моментами по данному продукту.
* Создать тест на проверку знаний.

**3. Используемые инструменты:**

* PowerPoint
* Veeva
* Docsfera
* MS Form

**4. Описание результата:**

* Медпредставитель получает сформированный учебный материал, основанный на истории промоции продукта.
* ИИ выделяет ключевые моменты, на которые стоит обратить внимание, и подсказывает верные формулировки для общения с врачом.
* По итогам прохождения курса ИИ создает проверочный тест.

**5. Эффективность решения:**

* Отсутствие шаблонных презентаций.
* Сокращение времени на подготовку материалов для введения в должность нового сотрудника или запуска нового продукта.
