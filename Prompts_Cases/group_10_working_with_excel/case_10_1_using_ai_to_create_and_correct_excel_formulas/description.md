**Пример 2: Исправление существующей формулы**

```
Помоги мне исправить формулу в Excel в колонке H. Мне нужно определить регион для каждого сотрудника для таблиц "Продажи" и "Цели".

У меня есть 3 таблицы:

1. Продажи по регионам и продуктам.
2. Целевые показатели.
3. Закрепление региона по сотрудникам.
Ответ дай на русском, формулы на английском языке, с разделителем ";".

```
