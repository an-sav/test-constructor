#### **Кейс 10.2: Использование ИИ для создания меппингов (сопоставления данных)**

**2. Шаги выполнения:**

1.  Скопируйте текст промпта в Concierge, но не отправляйте.
2.  Добавьте данные, которые нужно сопоставить. Это можно сделать двумя способами:
    * Прикрепить Excel-файл со списками.
    * Вставить списки непосредственно в диалоговое окно с промптом.
3.  Отправьте запрос.
4.  Получите результат в виде таблицы, скопируйте его в Excel и проверьте.

**3. Используемые инструменты:**

* Concierge
* Excel

**4. Описание результата:**

* Готовая таблица с сопоставленными данными (мэппинг).
* Информация отсортирована согласно запросу.
* Выделены пробелы и несоответствия, что упрощает дальнейшую обработку.
* Полученный мэппинг можно сохранить и использовать многократно.

**5. Эффективность решения:**

* **Экономия времени:** ~75%. На 100 записей вместо 1+ часа ручной обработки потребуется около 15 минут.
* **Снижение ошибок:** Автоматизация процесса значительно снижает риск ручных ошибок.
* **Масштабируемость:** Легко применяется для больших объемов данных и может быть повторно использован для аналогичных задач.
