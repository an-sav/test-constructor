#### **Кейс 10.3: Использование ИИ для поиска формул в Excel, подбора стат. анализа и тестов**

**2. Шаги выполнения:**

1.  **Подготовка данных:** Соберите необходимые данные (рекомендуется до 500 значений) и нормативные документы, описывающие методологию расчета.
2.  **Запрос:** Введите промпт (Вариант 1 с данными или Вариант 2 с прикрепленным документом).
3.  **Проверка результатов:** Сверьте количество обработанных данных с исходным. При необходимости попросите ИИ сделать перерасчет, уточнив детали: `"В данных N значений, сделай перерасчет согласно прикрепленным документам"`.
4.  **Уточнение:** Задайте дополнительные вопросы, если что-то непонятно, например: `"Можешь объяснить, почему выбран этот метод расчета?"`.
5.  **Документирование:** Скопируйте результаты расчетов или готовую инструкцию в рабочий документ.

**3. Используемые инструменты:**

* Concierge
* Excel

**4. Описание результата:**

* **Вариант 1:** Проведенный анализ набора данных с рассчитанными статистическими показателями (например, контрольными лимитами) и формулами, которые использовались для расчета.
* **Вариант 2:** Подробная, структурированная инструкция для Excel, адаптированная под конкретный нормативный документ и тип данных.

**5. Эффективность решения:**

* **Сокращение времени:** Ускоряет анализ данных и подбор правильных статистических методов с нескольких дней до нескольких часов.
* **Автоматизация:** Помогает автоматизировать процесс подбора необходимых формул и формирования плана обработки данных, который затем можно использовать в валидированных программах.
* **Стандартизация:** Позволяет создавать стандартизированные подходы к расчетам.
