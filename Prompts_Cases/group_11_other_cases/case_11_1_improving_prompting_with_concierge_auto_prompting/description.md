#### **Кейс 11.1: Улучшение промптинга с помощью Консьерж (автопромптинг)**

**2. Шаги выполнения:**

1.  Подготовьте документ с рекомендациями или лучшими практиками по написанию промптов.
2.  Сформулируйте ваш первоначальный, простой промпт.
3.  Используйте указанный выше "мета-промпт", прикрепив к нему файл с лучшими практиками и вставив свой простой промпт.
4.  Отправьте запрос в Concierge.
5.  Ответьте на уточняющие вопросы, которые задаст Concierge, чтобы обогатить контекст.
6.  Получите от Concierge улучшенную версию вашего промпта и итоговый документ, созданный на его основе.

**3. Используемые инструменты:**

* Concierge
* Файл с лучшими практиками промптинга (в примере — 30-страничный документ).

**4. Описание результата:**

* Сначала Concierge, проанализировав простой промпт, предлагает серию **уточняющих вопросов** для его улучшения (о цели, аудитории, темах, тоне, формате и т.д.).
* После получения ответов Concierge формирует **структурированный, детальный промпт**.
* На основе этого улучшенного промпта создается **комплексное руководство**, включающее техники адаптации, рекомендации по преодолению тревожности, советы по организации рабочего места, пошаговый план возвращения и раздел Q&A.

**5. Эффективность решения:**

* **Улучшение качества:** Итоговый документ становится значительно более качественным и полным благодаря структурированному промпту.
* **Экономия времени:** Создание профессионального руководства занимает несколько минут вместо нескольких часов.
* **Расширение видения:** Автоматически включаются аспекты, о которых пользователь мог не подумать (например, рекомендации для родителей).
* **Системный подход:** ИИ помогает системно подойти к решению задачи, а не просто дать ответ на поверхностный запрос.
* **Обучение:** Пользователь учится правильно и детально формулировать запросы для получения лучших результатов.
