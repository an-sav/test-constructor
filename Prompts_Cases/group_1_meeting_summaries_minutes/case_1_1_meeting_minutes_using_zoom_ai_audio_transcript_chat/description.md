# Кейс 1.1 — Meeting minutes / Итоги бизнес‑встреч с использованием Zoom AI (аудио, транскрипт, чат)

## 2. Шаги выполнения

1. Активируйте вначале встречи AI‑ассистент Zoom (доступно владельцу встречи) — нажмите соответствующую кнопку.
2. После окончания встречи владельцу придет письмо с адреса `` с темой «Сводка конференции (название вашей встречи zoom)».
3. Откройте Concierge. В диалоговое окно введите промпт, но не отправляйте (не нажимайте Enter).
4. Откройте письмо от Zoom и скопируйте в Concierge текст сводки конференции (Ctrl+A, Ctrl+C). Вставьте скопированный текст в окно с промптом. Отправьте.
5. Получите резюме встречи. При необходимости задайте уточняющие вопросы. Если базовый язык — английский, можно использовать фразу «переведи на русский».
6. Поделитесь результатом с коллегами.

## 3. Используемые инструменты

- Zoom AI Companion
- Concierge

## 4. Описание результата

- Готовый структурированный документ с итогами встречи, соответствующий внутренним стандартам.

## 5. Эффективность решения

- Требуется около 5–7 минут на создание итогов вместо просмотра полноценной записи встречи.
