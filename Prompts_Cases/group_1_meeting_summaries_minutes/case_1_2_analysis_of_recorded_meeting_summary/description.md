**2. Шаги выполнения:**

1.  После окончания встречи получите на почту письмо от Zoom c названием "Zoom Recording". Перейдите по ссылке, чтобы попасть на страницу с файлами записи.
2.  Выберите нужную встречу и скачайте файл `Audio Transcript`.
3.  Откройте Concierge. В диалоговое окно введите промпт, но не отправляйте.
4.  Откройте файл `Audio Transcript` с помощью Notepad (или другого текстового редактора). Выделите всё (Ctrl+A), скопируйте (Ctrl+C) и вставьте в окно Concierge после текста промпта. Отправьте.
5.  Получите резюме встречи. При необходимости задайте уточняющие вопросы (например, если знаете о деталях, которые могли быть упущены).
6.  Поделитесь результатом с коллегами.

**3. Используемые инструменты:**

* Zoom Recording
* Concierge
