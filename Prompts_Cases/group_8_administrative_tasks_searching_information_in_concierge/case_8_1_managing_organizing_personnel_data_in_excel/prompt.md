**1. Решение (промпт):**

&gt; *Ниже несколько примеров промптов для разных задач.*

**Пример 1: Поиск сотрудников по отделу**

```
Напиши информацию о сотрудниках, работающих в отделе Digital в Санофи Россия.

```

**Пример 2: Структурирование информации в таблицу**

```
Представь информацию об этих сотрудниках в табличном виде, указав: Имя и Фамилия, Департамент, Должность, Контактные данные, и Колонку с информацией о проектах/процессах/системах, за которые отвечает сотрудник в компании.

```

**Пример 3: Дополнение существующего списка**

```
Дополни список из excel на основе имеющихся полей в таблице и добавь дополнительный столбец, в котором указан руководитель сотрудника.

```

**2. Шаги выполнения:**

1.  Сформулируйте запрос к Concierge для поиска информации о сотрудниках (например, по отделу или проекту).
2.  Проверьте полученный список. При необходимости попросите его дополнить.
3.  Используйте второй промпт, чтобы попросить Concierge структурировать полученную информацию в таблицу с нужными вам колонками.
4.  Скопируйте сгенерированную таблицу и вставьте ее в Excel или Word для дальнейшей работы.

**3. Используемые инструменты:**

* Concierge

**4. Описание результата:**

* Сформированная таблица со списком сотрудников и всей необходимой информацией в нужных полях (ФИО, должность, контакты, руководитель и т.д.).
* Таблицу можно легко использовать для формирования списков рассылок или для других административных и проектных задач.
