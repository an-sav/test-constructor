#### **Кейс 8.2: Получение информации из OneSupport (Snow)**

**2. Шаги выполнения:**

1.  Откройте окно Concierge.
2.  Введите простой промпт: "Покажи мои задачи в OneSupport".
3.  Concierge выведет список ваших открытых задач.
4.  Нажмите на ссылку нужной задачи, чтобы открыть ее напрямую в браузере.

**3. Используемые инструменты:**

* Concierge

**4. Описание результата:**

* Результатом выполнения промпта является отображение списка ваших задач из OneSupport (раздел "Мои задачи").
* Для удобства сразу отображаются прямые ссылки на каждую форму задачи.

**5. Эффективность решения:**

* **Обычный маршрут:** Зайти в браузер -&gt; открыть OneSupport -&gt; перейти в "Мои задачи" -&gt; найти нужную задачу в списке -&gt; открыть задачу.
* **С помощью Concierge:** Открыть окно Concierge -&gt; ввести промпт -&gt; нажать на нужную ссылку в полученном списке.
* Это значительно упрощает и ускоряет доступ к задачам, особенно если Concierge постоянно открыт, а уведомлений из OneSupport приходит много.
